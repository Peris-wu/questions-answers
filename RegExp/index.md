# 正则表达式

## 字符
```
  [abc]: 字符集,匹配集合中所含任一的字符
  [^abc]: 否定字符集，匹配集合中所含字符除外的字符
  [a-z]: 字符集范围,匹配指定范围内的任意字符
     . : 匹配除换行符外的任意单个字符
     \ : 转移字符
     \w: 匹配任何数字、字母和下划线
     \W: 匹配除数字、字母和下划线外的任意字符
     \d: 匹配任何数字
     \D: 匹配除数字外的任意字符
     \s: 匹配任何空白字符
     \S: 匹配除空白字符外的任何字符
```

## 数量表示
```
    ?: 匹配前面表达式0个或1个
    +: 匹配前面表达式至少1个
    *: 匹配前面表达式0个或多个
    |: 或运算符，匹配符号前后的表达式
    {m}: 匹配前面表达式m个
    {m,}: 匹配前面表达式至少m个
    {m,n}: 匹配前面表达式至少m个,最多n个
```

## 特殊标志
```
    //i: 忽略大小写
    //g: 全局匹配
    //m: 多行修饰符,用于多行匹配
```

## 分组
```
    (expression): 分组,匹配括号里的整个表达式。
    let reg = /_(\d)/
    let str = 'abc_125-abc_35'
    reg.exec(str) // output: _1 1 _3 3 
    (?:expression): 非捕获分组,匹配括号里的整个字符串但不获取匹配结果，拿不到分组引用。
```
